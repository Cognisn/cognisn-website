Spark

A secure, multi-provider AI research platform with a modern web interface. Chat with Claude, Gemini, Grok, Llama, and Mistral — with MCP tool integration, persistent memory, and autonomous scheduled actions.

Overview

Spark is a personal AI research kit that brings together multiple LLM providers under a single, clean web interface. Rather than switching between different AI chat services, Spark lets you work with all of them in one place — with powerful additions like tool use, persistent memory, and scheduled autonomous actions.

It's built with security in mind: API keys are stored in your OS keychain (never in config files), tool usage requires explicit permission, and a prompt inspection system watches for potential threats.

Supported Providers

  • Anthropic — Claude models
  • AWS Bedrock — Managed Claude, Llama, Mistral, and more
  • Google Gemini — Gemini models
  • Ollama — Local models (Llama, Mistral, etc.)
  • X.AI — Grok models

Features

Conversations

  • Real-time token streaming via Server-Sent Events
  • Dark and light theme with persistent preferences
  • LLM-driven context compaction when approaching token limits
  • Cross-conversation context sharing and linking
  • Favourites for quick access to important conversations

Tools & MCP Integration

  • Full Model Context Protocol support via stdio, HTTP, or SSE
  • Built-in tools for filesystem access, document processing (Word, Excel, PDF, PowerPoint), web search, and archive handling
  • Per-conversation tool enable/disable controls
  • First-use permission prompts with granular approval

Persistent Memory

  • Stores facts, preferences, project context, instructions, and relationships
  • Vector embedding-based semantic search for relevant recall
  • Automatic memory injection into conversation context

Autonomous Actions

  • Schedule tasks on cron or one-off schedules via APScheduler
  • AI-assisted action creation
  • Background daemon with system tray support (macOS/Windows)
  • Full execution history tracking

Security

  • API keys stored in OS keychain — never in config files
  • Pattern and keyword-based threat detection on prompts
  • Password-protectable settings page
  • Per-conversation, per-tool permission system

Download & Install

Spark ships with pre-built installers that include a bundled Python runtime — no dependencies required. Just download and run. Alternatively, install from PyPI if you prefer to manage your own Python environment.

Getting Started

Launch Spark with a single command:

spark

On first run, Spark will create a configuration file, start a web server on a random available port, and open your browser for initial setup. From there, configure your LLM providers and start chatting.

Configuration Locations

  • macOS: ~/Library/Application Support/spark/
  • Linux: ~/.config/spark/
  • Windows: %APPDATA%/spark/

Keyboard Shortcuts

  • Ctrl/Cmd + K — Go to Conversations
  • Ctrl/Cmd + N — New Conversation
  • Ctrl/Cmd + , — Open Settings
  • Enter — Send message
  • Shift + Enter — New line

Architecture

Spark is built on FastAPI with a Bootstrap 5 frontend using the Cognisn design system. The backend manages conversation context with compaction and memory retrieval, routes requests to LLM providers, and orchestrates MCP servers and built-in tools. Settings and secrets management is handled by Konfig.
